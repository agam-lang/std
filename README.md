# Agam Standard Library

> Part of the [agam-lang](https://github.com/agam-lang) organization.

## Mission

Provide the foundational standard library for the Agam language, covering collections, I/O, math, concurrency, networking, and tensor operations.

## Key Areas

- Collections
- I/O
- Math/Numerical
- Concurrency
- Networking
- String Processing
- Tensor Operations

## Status

This repository is under active development as part of the Agam ecosystem.
